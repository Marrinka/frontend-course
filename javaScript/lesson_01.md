# Массивы в JavaScript

**Задание:**
Написать конспект по теме "Массивы в JavaScript". 

Основные моменты, на которые стоит обратить внимание:
- Внутреннее устройство массива
- Push, pop, unshift, shift
- Перебор элементов

- sort и reverse
- split и join
